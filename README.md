# Web Development Portfolio - Activity Compilation

This project is a compilation of activities combined into a single application for the Web Development course, focusing on **PHP Laravel, Blade, and Bootstrap**.

## 🛠️ Stack
- **Backend**: PHP & Laravel 12
- **Frontend**: Blade & Bootstrap 5
- **Database**: SQLite

## 📦 Setup & Running

### Installation
```bash
composer run setup
```

### Run Locally
```bash
composer run dev
```
The server will run at [http://127.0.0.1:8000](http://127.0.0.1:8000).
